# \## Configuration File

# 

# \### Understanding `config.default.php`

# 

# `config.default.php` is the default configuration file for sWADAH.  

# \*\*Do not alter\*\* the default values directly.  

# Instead, make a copy and rename it to `config.user.php`.

# 

# Since sWADAH 2021X and all future versions, `config.default.php` contains comprehensive documentation on each setting.

# 

# \### Understanding `config.user.php`

# 

# You may create a `config.user.php` file that exists \*\*alongside\*\* `config.default.php`.  

# You can override any default value there. When upgrading sWADAH, `config.user.php` ensures your custom settings remain unchanged.

# 

# If you do \*\*not\*\* have this file, you’ll have to reconfigure everything again — so it is highly recommended to use it.  

# Settings in `config.user.php` \*\*overwrite\*\* those in `config.default.php`.
