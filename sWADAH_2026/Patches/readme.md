# sWADAH 2026 Patch:

**Patching instruction:**

1. Download any patch and make sure to follow the file name instructions.
2. For example, if the patch file name consists of patching from Release5 to Release6, simply overwrite files/folders given to your sWADAH directory.
3. If the patch is for database, after uncompressing there will be a .sql file. Apply the .sql to your sWADAH database inside MariaDB in order to patch to new structure.
